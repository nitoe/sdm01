Sistem Operasi Windows
Desktop · File Explorer · Manajemen File & Folder · Shortcut Keyboard
20%
Bobot: 20% Manajemen file & folder Shortcut keyboard Navigasi antarmuka
🖥️ Mengenal Desktop Windows
  • Desktop — layar utama setelah Windows menyala
  • Taskbar — bar bawah untuk beralih antar program
  • Start Menu — tekan ⊞ Win untuk membuka semua program
  • System Tray — kanan bawah, berisi jam, volume, Wi-Fi
  • File Explorer — buka dengan ⊞ Win+E
📁 Struktur Folder Windows
  • This PC — akses ke semua drive
  • Drive C: — tempat Windows dan program terinstal
  • Documents — folder default untuk dokumen kerja
  • Desktop — apa yang terlihat di layar utama
  • Downloads — file yang diunduh dari internet
📁 Membuat & Mengelola Folder
  • 1
    Membuat folder baru (Cara 1): Buka File Explorer atau Desktop → klik kanan di area kosong → pilih New › Folder → ketik nama → tekan Enter.
  • 2
    Membuat folder baru (Cara 2 — shortcut): Di File Explorer, tekan Ctrl+Shift+N → folder baru langsung siap diberi nama.
  • 3
    Mengganti nama (Rename): Klik folder sekali → tekan F2 → ketik nama baru → Enter. Atau klik kanan → Rename.
  • 4
    Menghapus: Pilih item → Delete → file masuk Recycle Bin. Untuk hapus permanen: Shift+Delete. Jika salah hapus, segera tekan Ctrl+Z.
📋 Copy, Cut, dan Paste File
  • 1
    Memilih file: Klik sekali untuk memilih satu item. Ctrl+Klik untuk memilih beberapa item. Ctrl+A untuk pilih semua.
  • 2
    Copy (Salin): Pilih file → Ctrl+C → pindah ke folder tujuan → Ctrl+V. File asli tetap ada di tempat semula.
  • 3
    Cut (Pindah): Pilih file → Ctrl+X → pindah ke folder tujuan → Ctrl+V. File asli akan hilang dari lokasi awal setelah di-paste.
  • 4
    Menjalankan program: Double-click ikon program di Desktop, atau tekan ⊞ Win → ketik nama program → tekan Enter. Beralih antar program: Alt+Tab.

💡 Tips Ujian — Sistem Operasi
  • Hafal Ctrl+C/V/X/Z/S sampai terasa otomatis — ini fondasi semua pekerjaan komputer.
  • Saat membuat folder, langsung ketik nama begitu folder terbuat — tidak perlu klik lagi.
  • Jika tidak sengaja menghapus file, tekan Ctrl+Z segera sebelum melakukan hal lain.
  • Cari program lebih cepat dengan tekan ⊞ Win lalu ketik nama program, daripada cari ikon di desktop.

Microsoft Word
Ukuran Kertas & Margin · Format Font · Paragraf · Insert Gambar
25%
Bobot: 25% Page setup Format font Paragraf Insert gambar
📄 Mengatur Ukuran Kertas & Margin
  • 1
    Ukuran kertas: Tab Layout › Size → pilih ukuran. Ukuran paling umum: A4 (21×29,7 cm) untuk dokumen standar Indonesia.
  • 2
    Margin: Layout › Margins → pilih preset (Normal, Narrow, Wide) atau Custom Margins... untuk angka tepat. Margin standar Normal: semua sisi 2,54 cm.
  • 3
    Orientasi: Layout › Orientation → Portrait (tegak) atau Landscape (rebah/horizontal).
  • 4
    Cara cepat: Klik tanda panah kecil (↘) di sudut grup "Page Setup" untuk membuka semua pengaturan halaman sekaligus dalam satu dialog.
🔤 Mengatur Format Font
  • 1
    Jenis font: Seleksi teks → klik kotak nama font di tab Home (biasanya "Calibri") → ketik atau pilih font. Shortcut: Ctrl+Shift+F.
  • 2
    Ukuran font: Seleksi teks → klik kotak angka ukuran → ketik angka → Enter. Atau: Ctrl+] (perbesar), Ctrl+[ (perkecil).
  • 3
    Bold / Italic / Underline: Seleksi teks → Ctrl+B (tebal), Ctrl+I (miring), Ctrl+U (garis bawah). Klik lagi shortcut yang sama untuk mematikannya.
  • 4
    Warna font: Seleksi teks → klik panah di sebelah tombol A (Font Color) di tab Home → pilih warna dari color picker.
  • 5
    Hapus semua format: Seleksi teks → klik tombol Clear All Formatting (ikon A dengan penghapus) di tab Home, atau tekan Ctrl+Space.

¶ Pengaturan Paragraf
  • 1
    Alignment (perataan): Ctrl+L = Rata kiri  ·  Ctrl+E = Rata tengah  ·  Ctrl+R = Rata kanan  ·  Ctrl+J = Justify (rata kiri-kanan). Untuk dokumen formal Indonesia biasanya menggunakan Justify.
  • 2
    Line Spacing (spasi baris): Seleksi paragraf → Home › Line and Paragraph Spacing (ikon garis dengan panah) → pilih angka (1.0, 1.5, 2.0) atau Line Spacing Options... untuk kustom.
  • 3
    Indentasi: Tab untuk menjorokkan ke kanan, Shift+Tab untuk mengurangi. Atau atur angka di Layout › Indent Left/Right.
  • 4
    Bullet & Numbering: Seleksi teks → klik tombol Bullets atau Numbering di tab Home. Klik panah kecil di sebelah tombol untuk memilih gaya bullet/nomor.
🖼️ Memasukkan Gambar & Mengatur Posisi
  • 1
    Insert gambar: Tab Insert › Pictures › This Device... → pilih file gambar → klik Insert.
  • 2
    Ubah ukuran gambar: Klik gambar → seret handle di sudut (bukan sisi) untuk mengubah ukuran proporsional tanpa distorsi. Untuk ukuran tepat: klik kanan → Size and Position....
  • 3
    Text Wrapping (tata letak gambar): Klik gambar → klik ikon Layout Options (muncul di kanan gambar) → pilih: In Line with Text (gambar seperti karakter teks), Square/Tight (teks mengelilingi gambar), Top and Bottom (teks di atas dan bawah), atau Behind/In Front of Text.
  • 4
    Memindahkan gambar: Ubah wrap ke selain "In Line with Text" → klik dan seret gambar ke posisi yang diinginkan. Gambar tidak bisa dipindah bebas jika masih "In Line with Text".
  • 5
    Crop gambar: Klik gambar → tab Picture Format › Crop → seret tepi hitam yang muncul → klik di luar gambar untuk selesai.
💾 Menyimpan Dokumen Word Ctrl+S untuk menyimpan cepat. Pertama kali: File › Save As → pilih lokasi → beri nama → pilih format .docx → klik Save. Biasakan simpan setiap beberapa menit agar tidak kehilangan pekerjaan.
💡 Tips Ujian — Microsoft Word
  • Atur page setup di awal sebelum mulai mengetik — lebih mudah dari mengubah di tengah pekerjaan.
  • Seleksi semua teks dengan Ctrl+A untuk mengatur font sekaligus jika diminta seragam.
  • Gambar tidak bisa dipindah bebas? Cek Text Wrapping — harus bukan "In Line with Text".
  • Gunakan Ctrl+Z jika format tidak sesuai — lebih cepat dari mengatur ulang manual.
  • Simpan dengan Ctrl+S setiap selesai satu bagian besar.

Bobot: 25% Membuat tabel data Lebar kolom & tinggi baris SUM · AVERAGE · MAX · MIN
📊 Mengenal Antarmuka Excel
  • Cell — kotak data, nama pakai kolom+baris (A1, B3, C10)
  • Row (Baris) — horizontal, bernomor (1, 2, 3...)
  • Column (Kolom) — vertikal, berhuruf (A, B, C...)
  • Name Box — pojok kiri atas, tampilkan alamat cell aktif
  • Formula Bar — bar di atas tabel, tampilkan isi cell
✏️ Navigasi & Input Data
  • Klik cell → ketik data → Enter (turun) atau Tab (ke kanan)
  • Edit isi cell: tekan F2 atau double-click
  • Hapus isi cell: pilih → tekan Delete
  • Pilih banyak cell: klik pertama → Shift+klik terakhir
  • Pilih seluruh kolom: klik huruf kolom di header
  • Pilih seluruh baris: klik angka baris di header kiri

  • 1
    Buat header tabel: Ketik nama kolom di baris pertama. Header biasanya dibuat Bold dan diberi background warna agar mudah dibedakan dari data.
  • 2
    Isi data: Gunakan Tab untuk bergerak ke kanan dan Enter untuk turun ke baris berikutnya. Masukkan data sistematis kiri ke kanan, atas ke bawah.
  • 3
    Merge & Center (gabung sel): Pilih beberapa cell → Home › Merge & Center. Sering untuk judul tabel yang mencakup beberapa kolom.
  • 4
    Border (garis tabel): Pilih semua cell tabel → klik panah di tombol Border → pilih All Borders untuk memberi garis di semua sisi.
  • 5
    Warna background: Pilih cell → klik panah di tombol Fill Color (ember cat) → pilih warna. Digunakan untuk membedakan header dari data.
↔️ Mengatur Lebar Kolom & Tinggi Baris
  • 1
    Seret batas kolom: Arahkan kursor ke batas antara dua huruf kolom di header hingga berubah menjadi panah dua arah (↔) → seret untuk mengubah lebar.
  • 2
    AutoFit lebar kolom: Double-click pada batas kolom di header → lebar otomatis menyesuaikan konten terpanjang. Atau klik kanan header kolom → Column Width... untuk masukkan angka tepat.
  • 3
    AutoFit semua sekaligus: Pilih semua (Ctrl+A) → klik kanan di header kolom mana saja → pilih Column Width, atau double-click batas header kolom mana saja.
  • 4
    Tinggi baris: Arahkan ke batas antara dua nomor baris di header kiri → seret. Atau klik kanan nomor baris → Row Height...
🔢 Rumus Dasar yang Wajib Dikuasai

Semua rumus Excel diawali dengan tanda =. Tanpa tanda ini, Excel menganggap isi cell sebagai teks biasa, bukan perhitungan.

SUM — Penjumlahan
=SUM(range)
=SUM(B2:B10)
Menjumlahkan semua angka dari B2 sampai B10. Bisa juga sel terpisah: =SUM(B2,C2,D2)
AVERAGE — Rata-rata
=AVERAGE(range)
=AVERAGE(C2:C10)
Menghitung rata-rata angka dari C2 sampai C10. Otomatis membagi total dengan jumlah data.
MAX — Nilai Tertinggi
=MAX(range)
=MAX(D2:D10)
Menemukan angka tertinggi dari D2 sampai D10. Berguna mencari nilai ujian tertinggi.
MIN — Nilai Terendah
=MIN(range)
=MIN(D2:D10)
Menemukan angka terendah dari D2 sampai D10. Berguna mencari nilai terendah dari data.
⚡ Cara Menggunakan Rumus — Step by Step (1) Klik cell tempat hasil akan ditampilkan (misal F2). (2) Ketik =SUM(. (3) Klik dan seret untuk memilih range (misal C2 sampai E2). (4) Ketik ). (5) Tekan Enter. Hasilnya langsung muncul. Untuk menyalin rumus ke baris bawah: klik cell F2 → arahkan ke pojok kanan bawah hingga muncul tanda + → seret ke bawah (AutoFill).
💡 Tips Ujian — Microsoft Excel
  • Selalu awali rumus dengan = — tanpa ini Excel tidak menghitung apapun.
  • Saat menulis rumus, pilih range dengan klik-seret langsung di spreadsheet — lebih cepat dan akurat dari mengetik alamat.
  • Rumus menampilkan #VALUE! atau #REF!? Cek apakah range benar dan semua cell berisi angka, bukan teks.
  • Gunakan AutoFit (double-click batas kolom) setelah mengisi data agar konten tidak terpotong.
  • Gunakan Fill Color untuk header agar tabel terlihat lebih rapi dan profesional.

Bobot: 30% — Terbesar Presentasi sistematis 6 slide Design & animasi Konsistensi tampilan
📑 Struktur Presentasi 6 Slide yang Sistematis

PowerPoint memiliki bobot terbesar (30%) karena menilai kemampuan menyampaikan informasi secara terstruktur dan visual. Presentasi yang baik harus memiliki alur yang jelas dari awal hingga akhir, dengan tampilan yang konsisten di seluruh slide.

Slide 1
🏷️ Cover / Judul
Judul presentasi besar dan jelas. Nama pembuat. Sub-judul jika perlu. Ini kesan pertama — buat menarik.
Slide 2
📋 Daftar Isi / Pendahuluan
Daftar topik yang akan dibahas, atau pengantar singkat. Memberi gambaran kepada penonton tentang isi presentasi.
Slide 3
📝 Konten Utama 1
Poin atau topik pertama. Gunakan bullet points singkat, bukan kalimat panjang. Tambah gambar/ikon relevan.
Slide 4
📝 Konten Utama 2
Poin atau topik kedua. Variasikan layout dari slide 3 agar tidak monoton — tapi warna dan font tetap konsisten.
Slide 5
📊 Elemen Visual
Gambar, tabel, diagram, atau infografis. PowerPoint bukan hanya teks — visual memperkuat pesan secara signifikan.
Slide 6
✅ Penutup / Kesimpulan
Ringkasan poin utama. Ucapan terima kasih. Boleh tambah pertanyaan diskusi atau call-to-action.
🎨 Menggunakan Fitur Desain (Design Theme)
  • 1
    Memilih tema: Tab Design → bagian "Themes" → pilih tema. Tema otomatis mengubah warna latar, font, dan elemen dekoratif seluruh presentasi sekaligus. Pilih tema di awal sebelum mengisi konten.
  • 2
    Variasi warna tema: Setelah pilih tema → di bagian "Variants" (kanan Themes) → pilih variasi warna. Mengubah palet warna tanpa mengganti desain dasar.
  • 3
    Format Background: Klik kanan area slide → Format Background... → pilih warna solid, gradien, gambar, atau tekstur. Centang "Apply to All" untuk seluruh slide.
  • 4
    Slide Layout: Klik kanan area slide → Layout → pilih tata letak (Title Slide, Title and Content, Blank, Picture with Caption, dll.).
  • 5
    Konsistensi adalah kunci: Satu tema untuk seluruh presentasi. Font, warna, ukuran elemen harus konsisten antar slide. Presentasi yang terlihat "campur aduk" menunjukkan kurangnya perencanaan.
✨ Animasi Objek & Transisi Slide
  • 1
    Menambahkan animasi: Klik objek (teks, gambar, dll.) → tab Animations → pilih efek (Appear, Fade, Fly In, dll.). Di bagian "Timing": atur Start (On Click / With Previous / After Previous), Duration, dan Delay.
  • 2
    Prinsip animasi baik: Gunakan animasi secara selektif, bukan semua objek. Animasi berlebihan justru mengganggu. Fade atau Appear menghasilkan tampilan paling bersih dan profesional.
  • 3
    Transisi antar slide: Klik slide di panel kiri → tab Transitions → pilih efek (Fade, Push, Wipe, dll.) → klik Apply to All untuk konsistensi di seluruh slide sekaligus.
  • 4
    Kecepatan transisi: Atur Duration di bagian Timing. Kecepatan ideal: 0,5–1 detik. Terlalu lambat (>2 detik) membosankan, terlalu cepat (<0,3 detik) terasa tergesa-gesa.
🖼️ Memasukkan Gambar di Slide
  • Insert › Pictures › This Device — dari file lokal
  • Klik gambar → seret handle sudut untuk resize proporsional
  • Tab Picture Format → Crop, Style, Color, Effects
  • Klik kanan → Send to Back / Bring to Front untuk atur layer
▶️ Mode Presentasi & Simpan
  • F5 — mulai dari slide pertama
  • Shift+F5 — mulai dari slide saat ini
  • → atau klik — ke slide berikutnya
  • ← — ke slide sebelumnya
  • Esc — keluar dari mode presentasi
  • Ctrl+S — simpan file .pptx
⚠️ Kesalahan Paling Umum dalam Ujian PowerPoint (1) Terlalu banyak teks di satu slide — setiap slide idealnya hanya 5–7 poin singkat, bukan paragraf panjang. (2) Font terlalu kecil — minimal 18pt untuk isi, 28pt+ untuk judul. (3) Warna teks tidak kontras dengan latar — teks gelap di latar gelap tidak bisa dibaca. (4) Setiap slide tampil berbeda tanpa tema — terlihat tidak profesional. (5) Animasi berlebihan dan tidak perlu — mengganggu, bukan memperindah.
💡 Tips Ujian — Microsoft PowerPoint
  • Pilih tema desain di awal sebelum mengisi konten — memastikan konsistensi dari slide pertama.
  • Rencanakan urutan 6 slide (Cover → Daftar Isi → Konten 1 → Konten 2 → Visual → Penutup) sebelum mulai mengetik.
  • Gunakan Ctrl+D untuk menduplikasi slide dengan cepat sambil menjaga konsistensi layout.
  • Tambahkan nomor slide: Insert › Slide Number — tampak profesional dan memudahkan navigasi.
  • Preview seluruh presentasi dengan F5 sebelum menyimpan — pastikan animasi dan transisi berjalan seperti yang diinginkan.
  • Simpan dalam format .pptx agar bisa diedit kembali. Gunakan Save As › PDF jika ingin mengunci tampilan.
